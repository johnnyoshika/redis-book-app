# Sample App to Experiment with Redis

## Setup Redis with Docker
* docker pull redis
* docker run --name redis-book-app -d -p 6379:6379 redis

## Run Application
* Clone this repo
* Run in Visual Studio

## View Redis Cache
* Download and install [Redis Desktop Manager](https://redisdesktop.com/)
* Connect with:
  * Address: localhost
  * Port: 6379
